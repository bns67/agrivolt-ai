# AgriVolt AI - MVP Platform
A centralized SaaS platform designed for agrivoltaics, predicting crop yields based on solar microclimates and providing AI-driven agronomy advice.

## Setup Instructions
1. **Backend:** Navigate to `/backend`. Run `pip install -r requirements.txt`.
2. **Start Server:** Run `uvicorn main:app --reload`. The API will run on `http://localhost:8000`.
3. **Frontend:** Navigate to `/frontend` and open `index.html` in your browser.
